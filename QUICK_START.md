# 🚀 快速开始指南

## 一、访问网站

### 方式1: 本地访问（云服务器内部）
```
http://localhost:3210
```

### 方式2: 公网访问（推荐）
```
http://你的云服务器公网IP:3210
```

**⚠️ 重要**: 确保云服务器安全组已放开 **3210端口** 的入站规则

---

## 二、使用流程（5步完成）

### 步骤1: 上传图片 📤
- 点击上传区域，或直接拖拽图片
- 支持格式: JPG, PNG, WEBP, GIF
- 文件大小: 最大10MB

### 步骤2: 选择拼豆规格 🎨
- **迷你豆** (2.6mm) - 适合精细作品
- **标准豆** (5.0mm) - 最常用规格 ⭐
- **大号豆** (10.0mm) - 适合儿童和大型作品

### 步骤3: 选择色卡档位 🎨
- **24色** - 基础色卡，适合简单图案
- **48色** - 扩展色卡，适合一般作品 ⭐
- **72色** - 完整色卡，适合复杂作品

### 步骤4: 调整画布尺寸 📏
- 宽度: 20-100颗（滑块调整）
- 高度: 20-100颗（滑块调整）
- 推荐: 50x50 起步 ⭐

### 步骤5: 开始转换 ✨
- 点击 **"✨ 开始转换"** 按钮
- 等待1-3秒完成转换
- 查看预览效果

---

## 三、高级功能

### 优化选项 🔧
- ✅ **移除孤立像素** - 消除噪点，推荐开启
- ✅ **平滑色块** - 减少颜色碎片，推荐开启

### 显示选项 👁️
- **网格线** - 显示/隐藏拼豆网格
- **色号** - 显示/隐藏色号标注（需要较大尺寸）
- **原图对比** - 叠加显示原图

### 缩放控制 🔍
- 滑块调整: 50% - 300%
- 方便查看细节

---

## 四、导出图纸 💾

### 1. PNG图纸
- 点击 **"📄 导出PNG"**
- 高清图纸，适合打印

### 2. PDF图纸
- 点击 **"📑 导出PDF"**
- 包含完整物料清单
- 适合打印和分享

### 3. 物料清单
- 点击 **"📊 导出清单"**
- CSV格式，可用Excel打开
- 包含色号、颜色名、用量

### 4. 分色图层
- 点击 **"🎨 分色图层"**
- 每种颜色单独一张图
- 适合分步制作

## 六、常见问题 ❓

### Q1: 无法访问网站？
**A**: 检查以下几点：
1. 服务是否启动: `netstat -tlnp | grep 3210`
2. 防火墙是否放开3210端口
3. 云服务器安全组是否配置正确

### Q2: 转换后颜色不准确？
**A**: 尝试以下方法：
1. 切换到更高档位色卡（48色或72色）
2. 开启"平滑色块"优化
3. 调整原图的亮度和对比度

### Q3: 转换速度慢？
**A**: 
1. 降低画布分辨率（如50x50）
2. 关闭"平滑色块"优化
3. 使用较小的原图

### Q4: 导出的图纸太小？
**A**: 
1. 增加画布分辨率
2. 导出后用图片查看器放大
3. 使用PDF格式导出（自动优化尺寸）

### Q5: 物料清单看不懂？
**A**: 
- 色号格式: P01-P72
- P01 = 白色, P14 = 黑色
- 数字越小越基础

## 七、使用技巧 💡

### 技巧1: 选择合适的原图
- ✅ 色彩鲜明的图片
- ✅ 对比度高的图片
- ✅ 主体清晰的图片
- ❌ 避免过于复杂的图片
- ❌ 避免色彩过渡太多的图片

### 技巧2: 参数调优
- 简单图案: 24色 + 30x30
- 一般作品: 48色 + 50x50 ⭐
- 复杂作品: 72色 + 80x80

### 技巧3: 分步制作
1. 导出分色图层
2. 按颜色逐层制作
3. 最后组合完成

### 技巧4: 保存常用配置
- 为不同类型作品创建预设
- 例如: "简单图案"、"人物肖像"、"风景照片"

## 九、示例工作流 📝

### 场景1: 制作简单Logo
```
1. 上传Logo图片（PNG透明背景）
2. 选择: 标准豆 + 24色
3. 设置: 30x30 画布
4. 开启: 移除孤立像素
5. 转换并导出PNG
```

### 场景2: 制作人物肖像
```
1. 上传清晰人物照片
2. 选择: 标准豆 + 72色
3. 设置: 60x60 画布
4. 开启: 移除孤立像素 + 平滑色块
5. 转换并导出PDF（含物料清单）
```

### 场景3: 制作风景画
```
1. 上传风景照片
2. 选择: 标准豆 + 48色
3. 设置: 80x60 画布
4. 开启: 平滑色块
5. 导出分色图层，分步制作
```

## 十、服务管理命令 🔧

### 查看服务状态
```bash
netstat -tlnp | grep 3210
```

### 查看服务日志
```bash
tail -f /tmp/vite.log
```

### 停止服务
```bash
pkill -f "vite.*3210"
```

### 重启服务
```bash
cd /root/perler-bead-generator
npm run dev > /tmp/vite.log 2>&1 &
```
